# PlainView

Windows용 가볍고 광고 없는 이미지 뷰어입니다.

[English README](README.md)

> 이미지를 열면 이미지 자체만 깔끔하게 표시합니다.
> 마우스를 움직이면 최소한의 오버레이 컨트롤이 나타납니다.

## 언어 지원

PlainView는 한국어와 영어 UI를 지원합니다. 시스템/브라우저 언어가 한국어이면 한국어를 사용하고, 그 외 환경에서는 영어를 사용합니다. 개발 중에는 로컬 개발 URL에 `?lang=ko` 또는 `?lang=en`을 붙이거나 `localStorage.plainview.locale` 값을 `ko` 또는 `en`으로 설정해 언어를 강제로 확인할 수 있습니다.

## 지원 형식

JPG, JPEG, PNG, WebP, GIF, BMP, TIF, TIFF, ICO, AVIF, JXL, PSD, TGA, PBM, PGM, PNM, PPM, PAM, DDS

JPG, JPEG, PNG, WebP, GIF, BMP, AVIF는 큰 base64 문자열로 복사하지 않고 WebView asset protocol로 직접 표시합니다.
설치된 WebView 런타임이 AVIF를 표시하지 못하면 PlainView는 AVIF 전용 오류를 명확히 표시합니다.

PAM 지원은 표준 tuple type 중심입니다.
DDS 지원은 DXT1/BC1, DXT3/BC2, DXT5/BC3 계열 중심입니다.
BC4, BC5, BC6H, BC7, 기타 DXGI 변형, 일반 비압축 DDS 파일은 지원하지 않습니다.

HEIC, HEIF, RAW, CR2, NEF, ARW 확장자는 인식하지만, 현재 버전에서는 열 때 지원하지 않는 형식이라는 오류를 명확히 표시합니다.

## 키보드 단축키

| 키 | 동작 |
|---|---|
| `Esc` | 닫기 |
| `Left Arrow` / `Backspace` | 이전 이미지 |
| `Right Arrow` / `Space` | 다음 이미지 |
| `+` | 확대 |
| `-` | 축소 |
| `0` | 원본 크기 |
| `F` | 화면 맞춤 |
| `T` | 항상 위 고정 전환 |
| `R` | 시계 방향 90도 회전 |
| `Ctrl+C` | 이미지 복사 |
| `Ctrl+S` | 다른 이름으로 저장 |
| `Ctrl+M` | 다른 폴더로 이동 |
| `Delete` | 휴지통으로 이동 |

## 기능

- **테두리 없는 창** - 이미지를 주 화면으로 유지하는 최소 UI
- **호버 오버레이** - 필요한 순간에만 반투명 컨트롤이 나타나고 2초 뒤 자동으로 숨김
- **폴더 탐색** - 같은 폴더의 이미지를 자동으로 목록화해 이전/다음 탐색 지원
- **확대/축소 컨트롤** - 마우스 휠, 키보드, 버튼으로 확대/축소
- **이미지 패닝** - 확대 상태에서 드래그로 이미지 이동
- **전체 화면** - 더블 클릭으로 전체 화면 맞춤, 다시 더블 클릭하면 이전 배율 복원
- **창 이동** - 기본 모드에서 이미지 영역 드래그, 확대 상태에서는 상단 핸들 또는 Alt-드래그 사용
- **컨텍스트 메뉴** - 이미지·파일 경로 복사, 탐색기에서 보기, 기본 앱으로 열기, 다른 이름으로 저장, 이름 바꾸기, 사용자 정의 앱 등록/열기, 휴지통 이동, 인쇄
- **이미지 복사** - `Ctrl+C` 또는 컨텍스트 메뉴로 현재 이미지를 클립보드에 복사
- **파일 경로 복사** - 컨텍스트 메뉴에서 현재 이미지의 전체 경로를 클립보드에 복사
- **다른 이름으로 저장** - `Ctrl+S` 또는 컨텍스트 메뉴로 원본 이미지 파일 저장
- **이름 바꾸기** - 컨텍스트 메뉴에서 이미지 확장자를 유지한 채 파일 이름 변경
- **빠른 파일 이동** - 컨텍스트 메뉴 또는 `Ctrl+M`으로 현재 이미지를 다른 폴더로 이동
- **휴지통 지원** - 컨텍스트 메뉴 또는 `Delete`로 현재 이미지를 휴지통으로 이동
- **GIF 일시정지** - GIF를 클릭해 현재 프레임에서 일시정지하고, 다시 클릭해 재생
- **이미지 정보** - 하단 정보 바에 마우스를 올려 경로, 크기, 파일 크기, 확장자 확인
- **항상 위 고정** - 이미지를 다른 창 위에 유지
- **회전 및 직접 배율 입력** - 보기 전용 90도 회전과 하단 배율 라벨을 통한 숫자 배율 입력
- **드래그 앤 드롭** - 창에 이미지를 끌어 놓아 열기
- **설정 저장** - 창 위치, 항상 위 고정 상태, 관련 설정을 자동 저장

## 기술 스택

- [Tauri v2](https://tauri.app/) - 가벼운 데스크톱 앱 프레임워크
- [React 19](https://react.dev/) + TypeScript
- [Vite](https://vite.dev/) - 프론트엔드 빌드 도구

## 개발 환경

```text
Rust 1.94+, Node.js 20+, npm 10+
```

## 로컬 실행

```bash
# 의존성 설치
npm install

# 개발 모드 실행
npm run tauri dev

# 프로덕션 빌드
npm run tauri build

# Windows 릴리즈 산출물 생성: setup, MSI, portable EXE
npm run release:windows
```

## 현재 제한 사항

- 설치 후 Windows 기본 앱 설정에서 파일 연결을 수동으로 구성해야 합니다.
- PlainView는 보기 전용이며 이미지 편집 도구를 포함하지 않습니다.
- GIF 지원은 클릭으로 일시정지/재생하는 기능으로 제한됩니다. 프레임 단위 탐색은 지원하지 않습니다.
- 이미지는 base64로 로드되며 5개 이미지 LRU 캐시를 사용합니다.
